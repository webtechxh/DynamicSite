How to make Tournify work :-

Step 1 - Extract the zip into a directory of your choosing

Step 2 - Navigate to each of the subdirectories 'api' and 'reactapp' and open a terminal / cmd in them

Step 3 - In the command line, type "npm install"

Step 4 - Once all the dependencies are installed execute "npm start" in each of the respective terminals/cmd

Step 5 - Open the browser if it isn't automatically opened and go to http://localhost:3000/

Step 6 - Navigate to https://localhost:8443/ and accept use of the self-signed certificate in the same browser in which reactapp is running

Step 7 - Enjoy!
